BACKLOG DEL PROYECTO - SOLUCIÓN DE VEEDURÍA CIUDADANA

[ALTA PRIORIDAD]
1. Como ciudadano quiero registrar una incidencia de forma anónima para dejar evidencia del caso sin temor a represalias.
2. Como coordinador del proyecto quiero recibir una alerta automática en Telegram cada vez que se cree un reporte para revisarlo de inmediato.

[MEDIA PRIORIDAD]
3. Como miembro del equipo de veeduría quiero consultar la lista completa de reportes activos para hacerles un seguimiento estructurado.

[BAJA PRIORIDAD]
4. Como ciudadano quiero ver un tablero público con estadísticas de las incidencias resueltas para comprobar la transparencia del proceso.
